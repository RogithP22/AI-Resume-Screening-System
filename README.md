# AI Resume Screening System

## Overview

An AI-powered Resume Screening System built using Python, Streamlit, and NLP techniques to compare resumes against job descriptions.

## Features

- Upload Resume PDF
- Extract Resume Text
- ATS Match Score Calculation
- Skills Analysis
- Matching Skills Detection
- Missing Skills Detection
- ATS Recommendation

## Technologies Used

- Python
- Streamlit
- PyPDF2
- Scikit-Learn
- TF-IDF Vectorization
- Cosine Similarity

## How to Run

```bash
pip install -r requirements.txt
python -m streamlit run app.py
```
